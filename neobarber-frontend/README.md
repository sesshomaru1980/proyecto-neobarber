# NeoBarber Frontend

Frontend Angular del sistema NeoBarber.

## Requisitos
- Node.js LTS
- Angular CLI
- Backend NeoBarber ejecutándose en `http://localhost:3000`

## Instalación
```bash
npm install
npm start
```

## Módulos
- Home / landing
- Login y registro
- Navbar por roles
- Servicios (CRUD para Admin)
- Barberos (listar y desactivar para Admin)
- Mis citas
- Agendar cita con validaciones de horario
- Panel barbero (agenda, cambio de estado, bloqueo de horario)
